# راهنمای تست با XAMPP و آپلود وب‌مانیا

## نسخه فعلی

بک‌اند پروژه به MySQL وصل شده و دیگر از JSON برای ذخیره دیتا استفاده نمی‌کند.

## تست روی XAMPP

1. XAMPP را باز کن و این دو سرویس را Start کن:
   - Apache
   - MySQL

2. پوشه پروژه را داخل `htdocs` بگذار. مسیر پیشنهادی:

```txt
C:\xampp\htdocs\webmania
```

3. تنظیمات دیتابیس در این فایل است:

```txt
admin/config.php
```

تنظیم پیش‌فرض برای XAMPP:

```php
'db_host' => '127.0.0.1',
'db_name' => 'webmania',
'db_user' => 'root',
'db_pass' => '',
```

4. نصب دیتابیس را اجرا کن:

```txt
http://localhost/webmania/admin/install.php
```

این فایل دیتابیس `webmania` و جدول‌ها را می‌سازد.

5. سایت را باز کن:

```txt
http://localhost/webmania/
```

6. پنل مدیریت:

```txt
http://localhost/webmania/admin/
```

ورود پیش‌فرض:

```txt
username: admin
password: admin12345
```

## راه جایگزین با phpMyAdmin

اگر خواستی دستی ایمپورت کنی:

1. برو به:

```txt
http://localhost/phpmyadmin
```

2. از Import فایل زیر را وارد کن:

```txt
admin/database.sql
```

## روی هاست واقعی

1. فایل‌ها را در `public_html` آپلود کن.
2. در هاست یک دیتابیس MySQL بساز.
3. مشخصات دیتابیس را در `admin/config.php` وارد کن.
4. یک بار این آدرس را باز کن:

```txt
https://your-domain.com/admin/install.php
```

5. بعد از نصب، برای امنیت بهتر فایل زیر را حذف کن:

```txt
admin/install.php
```

6. حتماً رمز پنل را در `admin/config.php` تغییر بده.

## پوشه‌های قابل نوشتن

برای آپلود تصویر و فایل از پنل، این پوشه باید قابل نوشتن باشد:

```txt
admin/uploads/
```

اگر آپلود کار نکرد Permission را روی `755` یا در صورت نیاز `775` بگذار.

## نت ملی و سرعت

کتابخانه‌های اصلی فرانت‌اند داخل پروژه در پوشه زیر قرار داده شده‌اند و دیگر از CDN خارجی لود نمی‌شوند:

```txt
vendor/
```

تصاویر پیش‌فرض سایت هم لوکال هستند و برای نمایش اولیه به Unsplash یا CDN خارجی وابسته نیستند. فایل `.htaccess` روت هم برای کش و فشرده‌سازی اضافه شده است.

## پشتیبانی هوشمند AI

ویجت چت سایت از طریق بک‌اند PHP به سرویس AI وصل می‌شود و کلید API در فرانت‌اند قرار نگرفته است. تنظیمات در فایل زیر است:

```txt
admin/config.php
```

روی XAMPP/هاست باید افزونه `cURL` فعال باشد. محدودیت ارسال پیام و بلاک موقت برای اسپم/فحاشی در `admin/api.php` پیاده‌سازی شده است.
